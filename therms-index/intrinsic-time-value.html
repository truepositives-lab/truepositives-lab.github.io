Intrinsic vs Time Value

Understanding the components of option premium

Option Value Components

An option's premium consists of two components: intrinsic value and time value. Understanding these components is crucial for option trading and valuation.

Option Premium Decomposition

Option Premium = Intrinsic Value + Time Value

Call Intrinsic Value: max(S - K, 0)
Put Intrinsic Value: max(K - S, 0)
Time Value: Option Premium - Intrinsic Value

S = Current stock price, K = Strike price

Strike Selection Strategy

Buy ITM Options

✓ Pros

  • High intrinsic value
  • Less time decay risk
  • Higher delta (more stock-like)

✗ Cons

  • Higher premium cost
  • Lower leverage
  • Less upside potential

Buy ATM Options

✓ Pros

  • Balanced risk/reward
  • High gamma
  • Most liquid strikes

✗ Cons

  • Highest time value
  • Maximum theta decay
  • 50/50 chance of expiring ITM

Buy OTM Options

✓ Pros

  • Lowest premium cost
  • Highest leverage
  • Maximum % return potential

✗ Cons

  • 100% time value
  • High probability of total loss
  • Low delta (less responsive)

Key Concepts

Moneyness

ITM (In-the-Money): Has intrinsic value > 0

ATM (At-the-Money): Strike ≈ Current price (100% time value)

OTM (Out-of-the-Money): No intrinsic value (100% time value)

Time Value Factors

Time to expiration: More time = higher time value

Volatility: Higher volatility = higher time value

Interest rates: Higher rates increase call time value

Dividends: Expected dividends reduce call time value

Early Exercise

American Calls: Rarely optimal to exercise early (lose time value)

Exception: Deep ITM calls before ex-dividend date

American Puts: May be optimal when deep ITM
